Приложение-сервис для мониторинга состояний видеокарт Nvidia.

Имеет две части:

WPF составляющая и сервисную.

WPF выступает в качестве GUI интерфейса: Берет данные для отображения с помощью цикличных обращений к сервису, который работает в фоне.

Сервис, в свою очередь обращается к видеокарте Nvidia посредством библиотеки NvAPIWrapper.Net, которая представляет доступ к API.
